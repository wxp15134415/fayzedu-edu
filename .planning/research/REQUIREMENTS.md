# Fayzedu 教务管理系统需求规格

## 1. 项目概述

### 1.1 项目背景

Fayzedu 是一款面向教育机构的教务管理系统，提供学生、教师、班级、课程、成绩等核心教务功能的数字化管理能力。

### 1.2 技术约束

- **后端**: Rust + Axum + SQLx + SQLite
- **前端**: Vue3 + NaiveUI + UnoCSS
- **认证**: JWT 双令牌机制
- **密码加密**: Argon2

## 4. 安全需求

### 4.1 认证授权

- JWT 令牌有效期：Access Token 30min，Refresh Token 7d
- 密码使用 Argon2 加密
- API 权限校验基于角色

### 4.2 数据安全

- 敏感字段（身份证号、电话）脱敏返回
- SQL 注入防护（参数化查询）
- 日志不记录敏感信息

---

## 5. 数据质量需求

### 5.1 必填字段验证

| 实体 | 必填字段 |
|------|---------|
| 学生 | 学号、姓名、性别、年级、班级 |
| 教师 | 工号、姓名、性别、学历 |
| 课程 | 课程代码、课程名称、学分 |
| 成绩 | 学生、课程、学期、分数 |

### 5.2 业务规则验证

- 成绩范围：0-100 分
- 学号/工号格式：字母+数字组合
- 班级人数上限：60人/班

## 6. API 设计规范

### 6.1 响应格式

```json
// 成功
{
  "code": 200,
  "message": "success",
  "data": {}
}

// 失败
{
  "code": 400,
  "message": "错误描述",
  "errors": {}
}
```

### 6.2 错误码规范

| 范围 | 含义 |
|------|------|
| 200-299 | 成功 |
| 400-499 | 客户端错误 |
| 500-599 | 服务端错误 |

### 6.3 分页规范

```json
{
  "data": [],
  "pagination": {
    "page": 1,
    "page_size": 20,
    "total": 100
  }
}
```
