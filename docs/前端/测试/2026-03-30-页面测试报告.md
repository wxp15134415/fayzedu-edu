## 错误详情

### 1. API 404 错误

多个页面调用后端 API 返回 404，主要原因是后端服务未实现对应的 API 接口：

| 页面 | API 路径 | 说明 |
|------|----------|------|
| `/list/common-list` | `/proxy-url/userPage` | 用户列表接口未实现 |
| `/list/draggable-list` | `/proxy-url/userPage` | 用户列表接口未实现 |
| `/demo/dict` | `/proxy-url/dict/list` | 字典列表接口未实现 |
| `/setting/account` | `/proxy-url/userPage` | 用户列表接口未实现 |
| `/setting/dictionary` | `/proxy-url/dict/list` | 字典列表接口未实现 |
| `/setting/menu` | `/proxy-url/getUserRoutes` | 菜单路由接口未实现 |

**错误信息**:
```
GET /proxy-url/userPage 404 (Not Found)
SyntaxError: Failed to execute 'json' on 'Response': Unexpected end of JSON input
```

### 2. 百度地图 API 错误

**页面**: `/demo/map`

**错误信息**:
```
百度地图 API 错误: APP 被禁用
TypeError: Cannot read properties of null
```

**解决方案**: 需要在百度地图开发者平台申请有效的 API Key

### 3. 外部资源加载失败

**页面**: `/demo/editor/md`

**错误信息**:
```
net::ERR_NAME_NOT_RESOLVED (via.placeholder.com)
```

**解决方案**: 检查网络连接或替换为可访问的图片资源

### 4. Content Security Policy 限制

**页面**: `/documents/vue`

**错误信息**:
```
Content Security Policy 阻止加载 iframe
```

**解决方案**: 需在后端配置 CSP 响应头允许 iframe 加载
