## 2. 部署环境要求

| 软件 | 版本要求 | 备注 |
|------|----------|------|
| 操作系统 | Ubuntu 20.04+ / CentOS 8+ | 推荐 Linux |
| Node.js | >= 18.x | 使用 nvm 管理 |
| PostgreSQL | >= 14.x | 数据库 |
| Nginx | >= 1.18 | Web 服务器 |
| Git | >= 2.x | 代码版本管理 |

## 3. 服务器配置

### 3.1 系统配置

```bash
# 创建部署用户
sudo adduser deploy
sudo usermod -aG sudo deploy

# 配置 SSH 密钥
# (略)
```

### 3.2 安装 Node.js

```bash
# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc

# 安装 Node.js
nvm install 18
nvm use 18

# 安装 pnpm
npm install -g pnpm
```

### 3.3 安装 PostgreSQL

```bash
# Ubuntu
sudo apt update
sudo apt install postgresql postgresql-contrib

# 启动服务
sudo systemctl start postgresql
sudo systemctl enable postgresql
```

### 3.4 安装 Nginx

```bash
# Ubuntu
sudo apt install nginx

# 启动服务
sudo systemctl start nginx
sudo systemctl enable nginx
```

## 4. 项目部署

### 4.1 拉取代码

```bash
cd /home/deploy
git clone <repository-url> edusys
cd edusys
```

### 4.2 配置环境变量

```bash
# 后端配置
cp backend/.env.example backend/.env
vim backend/.env

# 内容示例
DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=postgres
DB_PASSWORD=your_password
DB_DATABASE=edusys
JWT_SECRET=your_jwt_secret
PORT=3000
```

### 4.3 安装依赖

```bash
# 后端依赖
cd backend
pnpm install
pnpm run build

# 前端依赖
cd ../frontend
pnpm install
pnpm build
```

### 4.4 数据库初始化

```bash
# 登录 PostgreSQL
sudo -u postgres psql

# 创建数据库
CREATE DATABASE edusys;

# 创建用户
CREATE USER deploy_user WITH PASSWORD 'your_password';
GRANT ALL PRIVILEGES ON DATABASE edusys TO deploy_user;
```

### 4.5 Nginx 配置

```nginx
# /etc/nginx/sites-available/edusys

upstream backend {
    server 127.0.0.1:3000;
}

server {
    listen 80;
    server_name your-domain.com;

    # 前端静态文件
    location / {
        root /home/deploy/edusys/frontend/dist;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # API 代理
    location /api {
        proxy_pass http://backend;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_cache_bypass $http_upgrade;
    }
}
```

```bash
# 启用配置
sudo ln -s /etc/nginx/sites-available/edusys /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

## 5. 进程管理

### 5.1 使用 PM2 管理后端

```bash
# 安装 PM2
sudo npm install -g pm2

# 启动后端
cd /home/deploy/edusys/backend
pm2 start dist/main.js --name edusys-backend

# 查看状态
pm2 status

# 查看日志
pm2 logs edusys-backend

# 开机自启
pm2 startup
pm2 save
```

## 6. 验证部署

### 6.1 检查服务状态

```bash
# 检查后端
curl http://localhost:3000

# 检查前端
curl http://localhost:5173

# 检查 Nginx
sudo systemctl status nginx
```

### 6.2 常见问题排查

| 问题 | 排查方法 |
|------|----------|
| 后端无法启动 | 检查 .env 配置和端口占用 |
| 数据库连接失败 | 检查 PostgreSQL 服务和用户权限 |
| 前端页面空白 | 检查 dist 目录和 Nginx 配置 |
| 502 错误 | 检查后端服务是否运行 |
