## 1. AI 助手

### 功能介绍

AI 助手是一个内置于网页的智能助手，可以通过自然语言帮助用户完成操作。

### 位置

页面右下角的悬浮按钮，点击可展开对话面板。

### 使用方法

1. 点击"AI助手"按钮打开面板
2. 在输入框中描述你想做的事情
3. 按 Enter 或点击发送按钮

### 示例指令

```
帮我添加一个学生
查看所有教师列表
导出成绩表
创建新班级
```

### 配置说明

如需修改 API 配置，编辑文件：
```
frontend/src/components/AIAssistant/index.vue
```

```typescript
agent = new PageAgent({
  model: 'minimax-m2.5-free',  // 可用模型: minimax-m2.5-free, qwen/qwen3.6-plus:free
  baseURL: '/v1',               // 通过 Vite 代理访问
  apiKey: 'fayzwxp',            // API 密钥
  language: 'zh-CN',
  instructions: {
    system: systemKnowledge      // 系统知识库
  }
})
```

### 可用模型

| 模型名称 | 说明 |
|----------|------|
| minimax-m2.5-free | MiniMax 免费模型 |
| qwen/qwen3.6-plus:free | 通义千问免费模型 |

## 2. Browser-Use MCP Server

### 功能介绍

Browser-Use MCP Server 让 Claude Code Agent 可以控制浏览器自动化执行任务。

### 用途

- 自动化测试
- 网页数据采集
- 浏览器操作演示
- 通过自然语言控制浏览器

### 启动方式

#### 方式一：手动启动

```bash
cd /Users/wangxiaoping/fayzedu/edusys
python3 scripts/mcp-server.py
```

#### 方式二：通过 Claude Code（需要配置）

在 `.mcp.json` 中配置后，Claude Code 会自动加载。

### 使用方法

在 Claude Code 对话中，直接告诉它：
- "打开浏览器访问 http://localhost:5173"
- "打开百度"
- "访问某个网站并截图"

### 配置文件

- **MCP Server**: `scripts/mcp-server.py`
- **配置**: `.mcp.json`

### 环境要求

- Python 3.11+
- browser-use 包: `pip install browser-use`
- Chrome/Chromium 浏览器

### 可用工具

| 工具名称 | 功能 |
|----------|------|
| browser_navigate | 打开浏览器访问指定 URL |

## 3. UI 主题

### 功能介绍

应用了 Page Agent 的 UI 风格，采用蓝紫渐变主题色。

### 新配色方案

| 元素 | 颜色 |
|------|------|
| 主色 | 蓝色 #58c0fc |
| 渐变 | 蓝 #58c0fc → 紫 #bd45fb |

### 深色模式

系统支持深色模式，可通过添加 `dark` class 启用：

```html
<html class="dark">
```

### 相关文件

- `frontend/src/styles/tokens.css` - 设计令牌
- `frontend/src/main.ts` - Element Plus 主题配置

## 4. 标签页导航

### 功能介绍

浏览器式标签页导航，首页固定为第一个标签，支持多页面切换。

### 使用方法

- 点击标签切换页面
- 点击 × 关闭当前标签
- 右键标签显示更多选项（关闭其他、关闭所有）
- 首页不可关闭

### 相关文件

- `frontend/src/stores/tabs.ts` - 状态管理
- `frontend/src/components/TabsView/index.vue` - 标签组件
- `frontend/src/layouts/MainLayout.vue` - 布局集成

## 常见问题

### Q: AI 助手无法响应

A: 检查 API 服务是否运行：
```bash
curl http://localhost:9015/v1/models -H "Authorization: Bearer fayzwxp"
```

### Q: MCP 浏览器无法启动

A: 确保已安装 browser-use：
```bash
pip install browser-use
```

### Q: 如何切换主题色

A: 编辑 `frontend/src/styles/tokens.css` 中的颜色变量。
