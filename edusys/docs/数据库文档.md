# 数据库文档

## 概述

EduSys 系统使用 PostgreSQL 数据库，采用 TypeORM 作为 ORM 框架。以下是完整的数据库表结构说明。

## ER 图

```
┌─────────────┐     ┌─────────────┐     ┌─────────────────┐
│    user     │────>│    role     │<────│ role_permission │<───│ permission
└─────────────┘     └─────────────┘     └─────────────────┘

┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   grade     │────>│    class    │────>│   student   │
└─────────────┘     └─────────────┘     └─────────────┘
                                              │
                                              ▼
                                       ┌─────────────┐
                                       │   score    │
                                       └─────────────┘
                                              │
                                              ▼
                                       ┌─────────────┐
                                       │   exam     │
                                       └─────────────┘

┌─────────────┐     ┌─────────────┐
│   subject   │<────│ system_info │
└─────────────┘     └─────────────┘
```

## 表结构

### 1. sys_user (用户表)

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | 用户ID |
| username | varchar(50) | UNIQUE, NOT NULL | 用户名 |
| password | varchar(255) | NOT NULL | 密码（加密存储） |
| real_name | varchar(100) | NOT NULL | 真实姓名 |
| role_id | bigint | FK | 关联角色ID |
| grade_ids | text | NULLABLE | 可管理的年级ID列表 |
| class_ids | text | NULLABLE | 可管理的班级ID列表 |
| student_id | bigint | NULLABLE | 关联学生ID（家长账号） |
| status | smallint | DEFAULT 1 | 状态：1启用 0禁用 |
| phone | varchar(20) | NULLABLE | 手机号 |
| email | varchar(100) | NULLABLE | 邮箱 |
| last_login_time | timestamp | NULLABLE | 最后登录时间 |
| created_at | timestamp | DEFAULT NOW() | 创建时间 |
| updated_at | timestamp | DEFAULT NOW() | 更新时间 |

### 2. sys_role (角色表)

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | 角色ID |
| role_name | varchar(50) | NOT NULL | 角色名称 |
| role_code | varchar(50) | UNIQUE, NOT NULL | 角色编码 |
| role_desc | varchar(255) | NULLABLE | 角色描述 |
| is_system | smallint | DEFAULT 0 | 是否系统角色：1是 0否 |
| created_at | timestamp | DEFAULT NOW() | 创建时间 |
| updated_at | timestamp | DEFAULT NOW() | 更新时间 |

**系统角色：**
- super_admin - 超级管理员
- admin - 管理员
- teacher - 教师
- student - 学生
- parent - 家长

### 3. sys_permission (权限组表)

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | 权限ID |
| permission_name | varchar(50) | NOT NULL | 权限名称 |
| permission_code | varchar(50) | UNIQUE, NOT NULL | 权限编码 |
| permission_desc | varchar(255) | NULLABLE | 权限描述 |
| created_at | timestamp | DEFAULT NOW() | 创建时间 |
| updated_at | timestamp | DEFAULT NOW() | 更新时间 |

### 4. sys_role_permission (角色权限关联表)

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | bigint | PK, AUTO_INCREMENT | ID |
| role_id | bigint | FK | 角色ID |
| permission_id | bigint | FK | 权限ID |

### 5. grade (年级表)

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | int | PK, AUTO_INCREMENT | 年级ID |
| gradeName | varchar(50) | NOT NULL | 年级名称（如：高一、高二、高三） |
| schoolYear | varchar(20) | DEFAULT '2025-2026' | 学年 |
| entranceYear | int | DEFAULT 2025 | 入学年份 |
| period | varchar(20) | DEFAULT '高中' | 学段：小学/初中/高中 |
| semester | varchar(20) | DEFAULT '第一学期' | 学期：第一学期/第二学期 |
| studentCount | int | DEFAULT 0 | 学生数 |
| classCount | int | DEFAULT 0 | 班级数 |
| status | int | DEFAULT 1 | 状态：1启用 0禁用 |
| description | varchar(500) | NULLABLE | 描述 |
| createdAt | timestamp | DEFAULT NOW() | 创建时间 |
| updatedAt | timestamp | DEFAULT NOW() | 更新时间 |

### 6. class (班级表)

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | int | PK, AUTO_INCREMENT | 班级ID |
| classNo | int | NULLABLE | 班级编号 |

| gradeId | int | FK | 所属年级ID |
| teacherId | int | NULLABLE | 班主任ID |
| studentCount | int | DEFAULT 0 | 学生人数 |
| status | int | DEFAULT 1 | 状态：1启用 0禁用 |
| createdAt | timestamp | DEFAULT NOW() | 创建时间 |
| updatedAt | timestamp | DEFAULT NOW() | 更新时间 |

### 7. student (学生表)

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | int | PK, AUTO_INCREMENT | 学生ID |
| studentNo | varchar(50) | NULLABLE | 学籍辅号 |
| studentId | varchar(50) | NULLABLE | 学籍号 |

| name | varchar(50) | NOT NULL | 姓名 |

| year2Class | varchar(50) | NULLABLE | 高二班级 |

| gender | varchar(10) | NULLABLE | 性别：男/女 |

### 8. subject (科目表)

| 字段名 | 类型 | 约束 | 说明 |

## 表之间关系

| 关系 | 主表 | 从表 | 说明 |
|------|------|------|------|
| 用户-角色 | sys_role | sys_user | 多对一，一个用户对应一个角色 |
| 角色-权限 | sys_permission | sys_role_permission | 多对多，通过中间表关联 |
| 年级-班级 | grade | class | 一对多，一个年级有多个班级 |
| 班级-学生 | class | student | 一对多，一个班级有多个学生 |
| 考试-成绩 | exam | score | 一对多，一次考试多条成绩 |
| 学生-成绩 | student | score | 一对多，一个学生多条成绩 |
