## 主要文件说明

### 后端核心文件

| 文件 | 说明 |
|------|------|
| main.ts | 应用入口，配置 CORS、全局前缀、验证管道 |
| app.module.ts | 根模块，配置数据库、所有业务模块 |
| entities/*.entity.ts | TypeORM 实体定义 |

### 前端核心文件

| 文件 | 说明 |
|------|------|
| main.ts | 应用入口，配置 Element Plus、绿色主题 |
| router/index.ts | Vue Router 路由配置 |
| utils/request.ts | Axios 请求封装，含拦截器 |
| stores/user.ts | Pinia 用户状态管理 |
| layouts/MainLayout.vue | 主布局组件（含侧边栏、头部） |
