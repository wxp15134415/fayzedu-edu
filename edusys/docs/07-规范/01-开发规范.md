# FayzEdu 教育管理系统 - 开发规范 v1.0

> 更新时间：2026-04-03

## 1. 代码规范

### 1.1 通用规范

- 使用 TypeScript 进行类型开发
- 使用 ESLint + Prettier 进行代码格式化
- 使用有意义的变量和函数命名
- 保持函数单一职责
- 适当添加注释说明复杂逻辑

### 1.2 命名规范

| 类型 | 命名方式 | 示例 |
|------|----------|------|
| 文件名 |  kebab-case | user.service.ts |
| 类名 | PascalCase | UserService |
| 函数名 | camelCase | getUserById |
| 变量名 | camelCase | userList |
| 常量名 | UPPER_SNAKE_CASE | MAX_COUNT |
| 接口名 | PascalCase | UserInfo |
| 枚举名 | PascalCase | UserStatus |

### 1.3 目录结构规范

```
src/
├── modules/           # 业务模块
│   └── user/
│       ├── user.module.ts
│       ├── user.controller.ts
│       ├── user.service.ts
│       ├── user.entity.ts
│       ├── dto/           # DTO 目录
│       │   ├── create-user.dto.ts
│       │   └── update-user.dto.ts
│       └── entities/      # 私有实体
├── common/           # 公共模块
│   ├── decorators/
│   ├── filters/
│   ├── guards/
│   └── interceptors/
└── config/           # 配置文件
```

## 2. Git 提交规范

### 2.1 提交信息格式

```
<type>(<scope>): <subject>

<body>

<footer>
```

### 2.2 Type 说明

| 类型 | 说明 |
|------|------|
| feat | 新功能 |
| fix | Bug 修复 |
| docs | 文档更新 |
| style | 代码格式 |
| refactor | 重构 |
| test | 测试相关 |
| chore | 构建/工具 |

### 2.3 提交示例

```
feat(user): 添加用户搜索功能

支持按用户名、真实姓名搜索用户

Closes #123
```

## 3. API 设计规范

### 3.1 URL 命名

- 使用小写字母
- 使用复数名词
- 使用连字符分隔

```
GET    /api/users          # 获取用户列表
POST   /api/users          # 创建用户
GET    /api/users/:id      # 获取单个用户
PUT    /api/users/:id      # 更新用户
DELETE /api/users/:id      # 删除用户
```

### 3.2 响应格式

```typescript
// 成功
{
  "code": 200,
  "message": "success",
  "data": { ... }
}

// 分页
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [...],
    "total": 100,
    "page": 1,
    "pageSize": 10
  }
}

// 错误
{
  "code": 400,
  "message": "错误信息",
  "data": null
}
```

## 4. 数据库规范

### 4.1 表命名

- 使用小写字母
- 使用下划线分隔
- 添加模块前缀

```
sys_user              # 系统用户表
sys_role              # 角色表
sys_permission        # 权限表
```

### 4.2 字段命名

| 类型 | 命名方式 | 示例 |
|------|----------|------|
| 主键 | id | id |
| 外键 | table_id | user_id |
| 时间 | created_at | created_at |
| 更新时间 | updated_at | updated_at |
| 状态 | status | status |

### 4.3 索引规范

- 主键自动创建索引
- 外键字段添加索引
- 常用查询字段添加索引
