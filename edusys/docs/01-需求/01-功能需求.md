# FayzEdu 教育管理系统 - 需求文档 v1.0

> 更新时间：2026-04-03

## 1. 项目概述

### 1.1 项目背景

FayzEdu 是一个面向教育培训机构的综合管理系统，用于管理学生、教师、课程、成绩等信息。

### 1.2 系统目标

- 提供完善的用户权限管理
- 支持多角色多权限配置
- 支持班级、年级管理
- 支持学生信息管理

## 2. 功能需求

| 用户列表 | 查看所有用户 | P0 |

| 用户编辑 | 编辑用户信息 | P0 |
| 用户删除 | 删除用户 | P1 |
| 密码重置 | 重置用户密码 | P1 |

### 2.2 角色管理

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 角色列表 | 查看所有角色 | P0 |
| 角色创建 | 创建新角色 | P0 |
| 角色编辑 | 编辑角色信息 | P0 |
| 角色删除 | 删除角色 | P1 |
| 权限分配 | 为角色分配权限 | P0 |

### 2.3 权限管理

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 权限列表 | 查看所有权限 | P0 |
| 权限创建 | 创建新权限 | P0 |
| 权限编辑 | 编辑权限信息 | P1 |
| 权限删除 | 删除权限 | P1 |

### 2.4 认证与授权

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 用户登录 | JWT 身份认证 | P0 |

## 3. 数据模型

### 3.1 用户 (sys_user)

| 字段 | 类型 | 描述 |
|------|------|------|
| id | bigint | 主键 |
| username | varchar(50) | 用户名（唯一） |
| password | varchar(255) | 密码（加密存储） |
| realName | varchar(100) | 真实姓名 |
| roleId | bigint | 关联角色 |
| gradeIds | text | 管理的年级ID列表 |
| classIds | text | 管理的班级ID列表 |
| studentId | bigint | 关联学生（可选） |
| status | tinyint | 状态（1:启用 0:禁用） |
| phone | varchar(20) | 手机号 |
| email | varchar(100) | 邮箱 |
| lastLoginTime | timestamp | 最后登录时间 |
| createdAt | timestamp | 创建时间 |
| updatedAt | timestamp | 更新时间 |

### 3.2 角色 (sys_role)

| 字段 | 类型 | 描述 |
|------|------|------|
| id | bigint | 主键 |
| roleName | varchar(50) | 角色名称 |
| roleCode | varchar(50) | 角色编码（唯一） |
| roleDesc | varchar(255) | 角色描述 |
| isSystem | tinyint | 是否系统角色 |
| createdAt | timestamp | 创建时间 |
| updatedAt | timestamp | 更新时间 |

### 3.3 权限 (sys_permission)

| 字段 | 类型 | 描述 |
|------|------|------|
| id | bigint | 主键 |
| permissionName | varchar(50) | 权限名称 |
| permissionCode | varchar(50) | 权限编码（唯一） |
| permissionDesc | varchar(255) | 权限描述 |
| createdAt | timestamp | 创建时间 |
| updatedAt | timestamp | 更新时间 |

### 3.4 角色权限关联 (sys_role_permission)

| 字段 | 类型 | 描述 |
|------|------|------|
| id | bigint | 主键 |
| roleId | bigint | 角色ID |
| permissionId | bigint | 权限ID |

## 4. API 接口

### 4.1 认证

| 方法 | 路径 | 描述 |
|------|------|------|
| POST | /api/auth/login | 用户登录 |
| GET | /api/auth/profile | 获取当前用户信息 |

### 4.2 用户管理

| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /api/users | 获取用户列表 |
| POST | /api/users | 创建用户 |
| GET | /api/users/:id | 获取用户详情 |
| PUT | /api/users/:id | 更新用户 |
| DELETE | /api/users/:id | 删除用户 |

### 4.3 角色管理

| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /api/roles | 获取角色列表 |
| POST | /api/roles | 创建角色 |
| GET | /api/roles/:id | 获取角色详情 |
| PUT | /api/roles/:id | 更新角色 |
| DELETE | /api/roles/:id | 删除角色 |

### 4.4 权限管理

| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /api/permissions | 获取权限列表 |
| POST | /api/permissions | 创建权限 |
| GET | /api/permissions/:id | 获取权限详情 |
| PUT | /api/permissions/:id | 更新权限 |
| DELETE | /api/permissions/:id | 删除权限 |

## 5. 角色说明

| 角色编码 | 角色名称 | 描述 |
|----------|----------|------|
| super | 超级管理员 | 系统最高权限 |
| admin | 管理员 | 系统管理权限 |
| teacher | 教师 | 教学管理权限 |
| student | 学生 | 基础权限 |

## 6. 非功能性需求

### 6.1 性能要求

- 页面加载时间 < 3秒
- API 响应时间 < 500ms

### 6.2 安全要求

- 密码加密存储（bcrypt）
- JWT Token 认证
- 接口权限验证
