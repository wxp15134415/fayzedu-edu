## 3. 测试用例

### 3.1 登录测试

```javascript
// 登录成功
POST /api/auth/login
Body: { username: "super", password: "123456" }
预期: { code: 200, data: { token: "..." } }

// 登录失败
POST /api/auth/login
Body: { username: "super", password: "wrong" }
预期: { code: 401, message: "用户名或密码错误" }
```

### 3.2 用户管理测试

```javascript
// 获取用户列表
GET /api/users
Headers: { Authorization: "Bearer <token>" }
预期: { code: 200, data: [...] }

// 创建用户
POST /api/users
Body: { username: "test", password: "123456", realName: "测试用户", roleId: 1 }
```
