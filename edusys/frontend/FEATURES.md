# EduSys 实用功能扩展建议

## 一、成绩分析报表

### 1. 基础统计
- **平均分计算** - 按班级、科目、学期限
- **及格率/优秀率** - 自定义分数阈值
- **最高分/最低分** - 快速了解分数范围
- **分数分布图** - 柱状图展示各分数段人数

### 2. 排名分析
- **总分排名** - 年级/班级排名
- **单科排名** - 各科目排名
- **进步排名** - 与上次考试对比排名变化

### 3. 报表类型
- **班级成绩单** - 班级学生成绩表
- **科目成绩表** - 各科目成绩对比
- **学生成绩趋势** - 历次考试成绩变化

---

## 二、报表打印/导出

### 1. PDF 导出
- 使用 `html2pdf.js` 或 `jspdf` 将页面转为 PDF
- 支持打印模板自定义

### 2. 打印功能
- 使用 `window.print()` 或 `print-js`
- 打印样式优化 `@media print`

### 3. 推荐库
```
pnpm add html2pdf.js jspdf
pnpm add print-js
```

---

## 三、高级筛选

### 1. 筛选器组件
- 多条件组合筛选
- 日期范围筛选
- 下拉多选筛选
- 快速筛选标签

### 2. 筛选模板
- 保存常用筛选条件
- 一键应用保存的筛选
- 筛选条件分享

### 3. 推荐方案
- 使用 Element Plus `el-select` 多选
- 自定义筛选面板组件

---

## 四、日志记录

### 1. 操作日志
- 记录增删改查操作
- 记录时间、操作人、操作内容
- 支持日志查询和导出

### 2. 登录日志
- 记录登录时间、IP、设备
- 异常登录提醒

### 3. 数据变更日志
- 记录重要数据修改历史
- 支持版本回溯

### 4. 实现建议
- 后端创建 log 表
- 使用 NestJS 拦截器统一记录

---

## 五、其他实用功能

### 1. 表格列设置
- 用户自定义显示/隐藏列
- 保存列配置到本地

### 2. 拖拽排序
- 使用 `vuedraggable` 实现
- 适用于班级排序、科目排序等

### 3. 数据导入增强
- 导入模板下载
- 导入预览和校验
- 导入进度显示

### 4. 快捷操作
- 键盘快捷键支持
- 右键菜单操作

### 5. 数据校验
- 表单实时校验
- 批量数据导入前校验
- 重复数据检测

---

## 六、推荐安装的依赖

```bash
# 图表
pnpm add echarts vue-echarts

# PDF
pnpm add html2pdf.js jspdf

# 打印
pnpm add print-js

# 拖拽
pnpm add vuedraggable@next

# Excel 增强
pnpm add xlsx

# 日期处理
pnpm add dayjs
```

## 七、优先级建议

| 优先级 | 功能 | 说明 |
|--------|------|------|
| P0 | 成绩统计 | 基础平均分、排名 |
| P1 | 报表打印 | 导出 PDF |
| P1 | 操作日志 | 记录关键操作 |
| P2 | 高级筛选 | 多条件过滤 |
| P2 | 列设置 | 用户自定义列 |
| P3 | 拖拽排序 | 手动调整顺序 |
